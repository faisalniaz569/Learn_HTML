## Editor:

I recommend text editor Brackets, Atom or VS Code.

All editor has Extension.

Brackets Extension:

1. Emmets,
2. Beautify, (command: ctrl + shift + L)
3. Autosave every edit! ,
4. Themes (view -> themes -> Brackets Light/ Brackets Dark),
5. HTML Skeleton,
6. Super Right Click,
7. Brackets Tree Icons,
8. Bootstrap Starter Template,
9. ACB Bootstrap Snippets,
10. HTML comment button,
11. Autoprefixer,
12. ExportPDF,
13. QuickFormTool,
14. colorHints,
15. Special Html Characters.

Atom:

Ctrl + , (settings)

Install packages - 

1. Atom-beautify,
2. atom-live-server,
3. autosave-onchange,
4. Emmet,
5. Color-picker,
6. file-icons-types,
7. pigments,
8. project-icons
