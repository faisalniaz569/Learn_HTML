## Editor:

I recommend text editor Brackets, Atom or VS Code.

All editor has Extension.

Brackets Extension:

1. Themes (view -> themes -> Brackets Light/ Brackets Dark/ New Moon/ new sun),(Font size)
2. HTML Skeleton,
3. Beautify (command: ctrl + shift + L),
4. Emmets,
5. Brackets Tree Icons,
6. HTML comment button,
7. C style comment button,
8. Autoprefixer,
9. Brackets Snippets (by edc),
10. CSSFier,
11. QuickFormTool
12. Autosave every edit!,
13. Html Special character (command: Alt + C)
14. ColorHints

** Bootstrap starter template,
** ACB bootstrap snippets

Atom:
